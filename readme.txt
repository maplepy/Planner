- Для роботи програми потрібно, щоб на ПК була установлена програма Python (інтерпретатор)
Для того, щоб запустити програму, потрібно:
1. Розархівувати архів;
2. Запустити консоль cmd.exe;
3. Перейти в каталог програми через команду cd;
4. Запустити програму командою: python main.py
